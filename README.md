# 🔮 More Enchants

[![Modrinth](https://img.shields.io/badge/Modrinth-More%20Enchants-00AF5C?style=flat&logo=modrinth)](https://modrinth.com/mod/moreenchants-7)
[![Minecraft](https://img.shields.io/badge/Minecraft-1.21.1-62B47A?style=flat&logo=minetest)](https://minecraft.net)
[![Loader](https://img.shields.io/badge/Mod%20Loader-Fabric-dbd0b4?style=flat)](https://fabricmc.net)
[![License](https://img.shields.io/badge/License-MIT-blue?style=flat)](LICENSE)

**More Enchants** — лёгкий серверный Fabric-мод, который снимает ванильное ограничение на максимальный уровень зачарований (5 → 7) и добавляет умную механику комбинирования чар в наковальне.

## ✨ Возможности

- Увеличивает максимальный уровень всех зачарований с 5 до 7
- Позволяет повышать уровень чар через наковальню (Острота V + Острота V = Острота VI)
- Работает с любыми предметами и зачарованными книгами
- Эффективность VI действительно копает быстрее, чем V
- Полностью серверный — игрокам не нужно устанавливать мод на клиент

## 📦 Установка

1. Установите [Fabric Loader](https://fabricmc.net/use/) для Minecraft 1.21.11
2. Установите [Fabric API](https://modrinth.com/mod/fabric-api)
3. Скачайте последнюю версию мода из раздела [Releases](https://github.com/MiNISOM/More-Enchants/releases)
4. Поместите `.jar` файл в папку `mods` вашего сервера или клиента
5. Готово!

## 🛠️ Как использовать

1. Положите предмет или книгу с чарами в левое окно наковальни
2. Положите предмет или книгу с **точно такими же чарами** (тот же ID, тот же уровень) в правое окно
3. Заберите результат — уровень повысился на 1
4. Повторяйте, пока не достигнете VII уровня
